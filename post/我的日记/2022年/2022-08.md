## 重构个人博客

第一件事是合并之前按天为单位书写的日记，最开始准备用Python，但一搜感觉还是Unix tool 比较方便：

```shell
for dic in `ls -d */`; do
		cd ${dic%%/*}; 
    	for file in `ls -tr *.md`; do
      	cat $file >> `stat -f "%Sc" -t "%Y-%m"`.md; 
      done
      sed -i '' 's/^# .*//' 2022-08.md 
      mv 2022-08.md ../${file:0 :7}.md 
		cd ..; 
		rm -rf $dic;
done 
```

Python 是编程语言中的极简主义，而一整套UNIX interface 则是极简主义

## 一直不能适应当地的工作节奏

和同学打电话抱怨工作时忽然意识到：我之前一直很内疚自己的听课障碍症（从小到大从来没听过课，仰头无法集中注意力超过5分钟）其实是自然演化的最优解，如果我真的像上班那样完全集中精力，早就累趴下了。

上学每天时长轻松超过12小时，但上班只有8小时。我上学每天只集中精力工作3~4小时左右（写作业），所以才能无痛撑这么多。上班每天集中精力7小时高强度读写，我一个月就受不了跑路了。

我猛然意识到，这么多年来我一直活在欧洲人的生活节奏里，完全不能融入当地“拼冲杀卷”的社区文化和“everyone strives to burn him/hersellf out”生活环境啊。

我其实也想自己能更“拼一点，冲一把，卷死它”，达成父母兄长亲戚朋友的要求，但是大脑实在做不到啊，到点就下班，下班就罢工，就像里昂的诊所一样。

现在终于释怀了，怎么开心怎么来吧，做不了的事不能强求。
