（三）在上初中那一年，他们在老家买了一个非常豪华的房子，算是一种乡土之情吧！要坐全景电梯上300层，然后再往下转一层才到。我第一次，也是唯一一次去的时候就迷路了，让我印象最深的是那个地下室，有一种夏威夷的装修风格，巨型的木板床，还是水床。现在想起来已经有七八年没去过了吧，房子已经卖掉了，但是以租赁的方式一直在租着，因为他们还没有放弃荣耀返乡的梦想。

一想到这我就非常惋惜，说，唉呀好好的东西给你们糟蹋了，那么好的房子被装修得那么烂！那真是我住过的最好的房子了！他们反过来怄气我：哎呀，好端端这么聪明帅气，人生被你混得这么烂！

笑死，明白了，瞬间就不敢说什么了我。

确实，很多时候我们都拿世界一流标准来要求别人，但是从来都没做到过，否则也不会混的这么烂。从这个角度上看，糟糕的婚礼，糟糕的装修，糟糕的人生，也是德性匹配了。

（四）老师让同学一道一道题目挨个讲解，轮到我时我却不记得要讲哪一题了，我胡乱选择了一个，但是幸运女神没有眷顾我。

（五）距离上次参加这样的考试已经过去了一千多天，我是指动真格的那种。

考完后我们坐在教室里，对绝大多数人来说，这已经是人生的最后一场考试了（研一的结课考试）

但大家还是假装像小学生一样乖乖坐在座位上假装看书复习（已经没有必要了喂！）

我站起身来：难道大家不应该狂欢吗？有一个狂热的同学问了我一个非常极端的问题，而且语法根本就不正确。无论政治立场如何，他们都很尊敬我，喜欢问我这类问题。

------

醒来以后我忽然领悟了，也许还不如就这样一辈子把我们关在一个小屋子里，做那些毫无意义的事情直到死更好呢？话说我们那样年复一年钻研一些非常domain specific 的数学题、物理题，不停地摧残我们的心灵是为了什么？难道是为了造光刻机吗？可是绝大多数人毕业之后，从事的工作只要会说会写就行了，但他们又因为被完全摧毁，连话都说不出来。

------

我之前从来没想过这个问题，因为这个问题都被大家说烂了，我觉得既没道理又有道理，就这么虚无主义地“let it go”了

直到我今天做的这个梦，这个梦的前半段是有好大一节课没上，但是回去就要参加一个数学考试。我的脑袋停留在我现在的状态，也就是空空无物，什么数学知识都没有，我看了一下练习册，要学习两个领域的一些细节知识，但是光有知识屁用不顶，主要是一些题目的解法。

## 优化uploadImage.sh

```shell
#!/bin/bash

output_dir="/Users/kounarushi/mycode/web-blog/public/.pic"
echo "Upload Success:"
for var in "$@"; do
    # Get the file extension
    filename=$(basename -- "$var")
    extension="${filename##*.}"

    # Generate a unique filename
    imgName=$(uuidgen)."${extension}"

    # Resize the image using sips
    sips --resampleWidth 1080 "$var" --out "$output_dir/$imgName" > /dev/null

    # Optimize the image based on the file extension
    if [ "$extension" = "png" ]; then
        # Use pngcrush for PNG files
        pngcrush -q -ow "$output_dir/$imgName"
    else
        # Use ImageOptim for other file types
        /Applications/ImageOptim.app/Contents/MacOS/ImageOptim "$output_dir/$imgName"
    fi

    # Print the file URL
    echo "file://$output_dir/$imgName"
done
```
