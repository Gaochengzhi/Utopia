# 2023-02

## 梦

（一）每个人都最后走向了自己的归宿，无法挣脱命运的牵绊。

村口戴着口罩的王阿姨问我的研究生是全日制的还是在职的，我也得知老同学最后去了他爸公司（京东？）打工。

我们很久之前都只想做一只自由自在的小鸟，家族和历史完全不能改变我们飞行的方向，现在呢？龙生龙凤生凤，完全被困在自己的命运之中，曾经那些裹着金色外衣的传奇奋斗故事褪色了，现实露出了它灰暗的一面，如一团双面胶一般把我们粘在大地上，尽管我们的双眼还仰望着天空。

（二）连续三年，高通胀和高失业率把地狱踩向深渊，受够了苦难的流浪者，站在潘地曼尼南（Pandemonium，撒旦的宫殿）的死人地狱第壹议会大厅的顶楼的直升机坪上，表达了祈求改变的愿望。但第一护卫队长兼总理大臣阿撒兹勒第拾柒仍然想粉饰太平，称游行队只是不听劝阻想要来朝见殿下的狂热粉丝，而撒旦竟也对地狱里积累的怨气一无所知。 

此时，天空中突然响起了一阵恐怖的雷声，紧接着又是一阵阵硫磺酸夹杂着雨炎火石（注：十六小地狱之叫唤地狱里有「雨炎火石」之刑，是向罪人落下起火的石块）的狂风暴雨，撒旦坐立不安，他问身边的侍卫：“他们一共有多少人？” 侍卫答道：“大约有五百人，陛下。” 撒旦感受到一阵澎湃的感情在自己的心中涌现，他对侍卫长说道：“天怎么能这么大的雨？显然这帮人都是真心想要来见我，我却又这样无情！快去准备我的车，我要去接见他们。”

硫酸像滴答滴答的雨点，敲打在车窗的玻璃上，黑夜如地狱般漆黑，偶尔闪烁的绝望的火焰，撒旦面无表情地坐在副驾驶，显然他不明白命运正在引导他走向一条未知的轨迹。

咔嚓一声，一道黑影闪过，玻璃窗碎裂，阿撒兹勒用他的长剑穿入撒旦的胸口，不过仅是浅擦，并不能致撒旦于死地。撒旦畏缩一下，用嘶哑的声音问道：“为什么要背叛我？” 同时他抽出一把匕首，朝着阿撒兹勒刺来，两个人在挡风玻璃上激烈地对抗。阿撒兹勒的目光带着悲凉和愤怒，他知道叛变失败了，至少是对他自己来说。他用尽最后的力气骂一句无关紧要的脏话，只有博格斯曼最年老的长者才能听懂，大意是：“你这个愚蠢至极的家伙，去死吧”，说着他便触发了身上的炸弹，瞬间，他把地狱般的黑暗和火焰一起拉到了死亡的深渊，改变了所有在场人的命运。

一场石破天惊的密谋震撼了「死人地狱」，但我无暇顾及，就被撒旦专车载着恐怖的死亡淹没着，湮没到了永恒白色光芒之中。

作为帝国第拾玖亲卫队的下士，我从来不清楚自己在「活人世界」上是怎么死的，但我却还清晰地记得在死人地狱里的一切：“我又死了，原来在地狱里也是会死的。”

我睁开双眼，周围有一些零星的老人和小孩，“死亡如此残酷，让这些孩童们永远都无法踏入成年，而老者则永远无法重现生机。幸亏我死得不早不晚”

“这里是活人地狱，比死人地狱更可怕，这儿没有秩序和感情，只有是一群无意识的无赖，你们为我而牺牲，但很抱歉，在这里我也只是一个平凡的陌生人。”撒旦对众人说道。

此处凄惨的大地以及锈迹斑斑、昏暗无光的海浪间没有任何生机——唯一明显的不同之处在天空上，原本阴暗无光的天幕上充斥着浅绿色的微光，而是有着暗光的浅绿色，好像发霉胶片扫出来的效果，其次是大海，虽然波涛汹涌，但是却没有任何声音，也没有风。

“这里没有空气” 我对自己说。

我们一行人像《网球馆的宣誓里的草民》那样挤进了一个废弃的室内游泳馆，我惊喜的发现我的初恋情人卡塔琳娜居然也在这里。当然，她对我曾经的爱慕一无所知。

空气里弥散着潮湿的小雨，腐蚀着我的皮肤，让人感觉到阴冷，游泳池里却弥散着一股热气。我跳下泳池，发现浑浊的深绿色池水尽是凶残的野兽出没，它们像鳄鱼扑食饮水的麋鹿那样向我冲来，我拿起旁边的板凳腿使劲将一个软体乌龟砸得稀烂。但是这块烂肉很快又像橡皮泥一样复原。

“死人地狱会死，但活人地狱却没有死亡” 我对自己说。

我又扑腾了两下，发现我能飞，而且速度不小。这让我马上得出了一个危险的结论：“成千上万生长在地狱的凶残野兽们怀有对我们的仇恨之心、无尽的行动能力，可能随时就要找上门来将我们撕咬得四分五裂、永久地折磨下去了”

我看到卡塔琳娜也脱光了衣服，准备下池子，凝视着她凹凸有致的身体，我心中忽然充满了一股不合时宜的欲望。我鼓起勇气像她张开双臂：卡塔琳娜，我们来做爱吧！马上那些野兽就会找到我们，这是我们最后能享受的快乐时光了。

但她还是很害羞，转过头，用衣服遮住身子，望着她旁边的闺蜜不说话。

（三）两个饭局，一个反思。 

在四川旅游，听着邻座的妈妈兴奋地讲他儿子的美好前途。现实中总是祝福的我，在梦里却忍不住生出一丝嫉妒之情，特别是听到我们来自同一所学校，我甚至不敢承认。 

在雨天的晚上，我在街边寻找能吃点东西的地方（在梦里永远在吃，或者来来回回地寻找吃的东西），大街上有各种豪车，和代驾这些豪车的骑各种电动折叠车的中年大叔。有几种特别灵巧的折叠车我很喜欢。

我来到@孤城 家开的早餐店，店里没人，但可以自助点餐（没有厨子点餐有什么用？）

我点了两瓶牛奶、一碗面和一个韭菜饼。

店家奶奶来了，最先开始以为我是偷吃贼，然后我让@孤城 算账，但他一边玩手机一边计算小数点三位后的单价，我觉得他会算错就一起算，但是手机计算器没电了，而我在梦里的计算能力已经不能执行多个四位数的加法心算了。

我忽然想到，身处文明C 的很多人崇拜文明B，但是厌恶文明A。是因为文明B 努努力还能够得到，但是文明A 高出B 一个量级，得不到就要毁灭，吃不葡萄就嫌葡萄酸。

而我，对文明A 的向往来自于对文明B 的自卑和焦虑，其实不一定真喜欢文明 A，高出一个量级的世界对它的情感只会有惊讶，没有别的。

我舅舅就讨厌上海A，因为他只能从巢湖C到合肥B。我爸讨厌美国A，因为他……

（四）我有一个绝活，能在一个教室大小的空间内输出超高精度的仿真空间，具体来说就是早读上课前的那几分钟。梦里其他的物体都是窄视角、低精度的，但唯独那个场景我能几乎完美还原出所有人的一举一动，而且是可交互的。

我能记住所有人的名字和性格，但醒来后连班花在哪个年级、什么名字都记不住了。

大概是小学吧，我像见老朋友那样和每个人打招呼，询问我现在坐哪个座位。因为小学每周座位随机打乱，如果请假太久没上课的话就不知道自己坐到哪里去了。

一旦和人聊天就破功了，好假，只有那个人是真实的，别的人都模糊化处理了。

（五）很感人的一个故事，关系到传统和亚文化的冲突、女同、贫困山区、带着刺的爱。很像电影《赎罪》但是关键情节记不得了，导致写出来有点莫名其妙，比如二女儿最开始为什么崇拜兰阿姨？最后一个镜头让我印象最深。就是兰阿姨因为流言蜚语被迫出走，房子空出来了，还在呀呀学步的三女儿和二哥分别到两边屋檐上挂起招租的牌子，山村里一切照旧。

你不得不承认过去三十年让很多人忽然拥有了向现代生活靠近的机会，这种转变某种程度上是史诗的。

（六）我没有身份证，经常在考试的时候检查身份，我就给保安看驾照和学生证，保安会首先很惊讶以为我是日本人，要出去特别处理，一会又回来发现只是简单没有带身份证罢了。

（七）英语补习班刚好在巴以交界和另两派武装斗争的势力范围内，有一天刚下课区域就都被封锁了，男士有被拉壮丁的风险。

场景模糊起来，变成了一幕哑剧，大意是，即便这世界上只剩下了五个人，也会分成左中右、极左极右。

（八）赶着去上音乐课，我不知道为什么要这样，专有教室在我看来完全是形式主义，我没觉得音乐教室里的音响效果有多么好。

我随着音乐（也不知道是广播里的还是耳机里的）一路飞奔，边跑边唱，“here is something, something”旁边来了一群稚气未脱的小学生，开始指出我“things”发音上的错误。

我开始没有理他们，假装没听到，但是他们不断重复，“应该是things，不是thins！” 我也不好意思再装傻，问他们是几年级，刚刚英语考试怎么样？“你们猜猜叔叔是几年级了？”

“你知道吗？其实英语并不是越大越难，有些时候小学英语考试就比研究生英语考试要难，因为考试这东西有时候是来区分智商的，小学的时候还不知道你是什么水平，但到了研究生，人们就知道你是傻逼了。”

首先，计算甲和乙两组数据的均值和标准差：

甲组均值 $\bar{x}_1 = \frac{1.4+1.8+3.0+0.1+2.2+1.5+2.0+0.3+0.5+1.9}{10} = 1.52$

乙组均值 $\bar{x}_2 = \frac{1.9+0.8+3.0-0.5+3.0+2.5-0.5+2.5+2.0+2.5}{10} = 1.825$

乙组标准差 $s_2 = \sqrt{\frac{\sum_{i=1}^{10}(x_i-\bar{x}_2)^2}{9}} = 1.241$

> 在昔日的维也纳，人们过着好日子，人们活得轻松、无忧无虑，北方的德国人带着恼怒和轻蔑俯视着我们这些多瑙河畔的邻居：这些人既不“能干”，也不保持严整的秩序，而是让自己好好地享受生活，享受美食，在节日和剧院里找到乐趣，还能创作无与伦比的音乐。维也纳人的确不具备德国人的那种“能干”——这种“能干”毕竟使其他民族的生存遭受了无比的痛苦和毁灭——也没有那种贪婪的“想要赶超一切他人”和“向前冲”的愿望，他们更愿意愉快地闲谈，愿意融洽地相处，愿意在安详愉快和放松的共处中，每个人都各得其所，于任何人并无不利。

> 维也纳的咖啡馆是一种非常特别的设施，在这个世界上没有什么是可以与之相比的。它实际上是一种民主性的会所，任何人只要买上一杯廉价的咖啡就可以加入其中，这笔小小的入场费让每位客人都可以好几个小时坐下，讨论、写作、打牌、处理邮件，尤其是可以享用诸多的报纸和杂志。一个比较好的维也纳咖啡馆，会有城里的全部报纸，而且不光是维也纳的报纸，还有整个德意志帝国的报纸，法国的、英国的、意大利的、美国的报纸，此外还有世界上最重要的文学和艺术评论期刊，如《法国水星评论》（Mercure de France）、《新观察》（Neue Rundschau）、《创作室》（Studio）、《伯灵顿杂志》（Burlington Magazine）。我们马上知道这世界上所发生的事情，知道每一本新出版的书，知道每一场随便在哪里的演出，我们比较不同报纸上的评论。奥地利人能够在维也纳的咖啡馆里大量地了解世界上发生的事情，同时能在好友圈子里讨论这些事情，在形成“奥地利人的思想活力和国际视野方面，也许没有什么能比咖啡馆的贡献更大。我们每天坐在那里几个小时，没有什么能逃过我们的注意力，因为我们的兴趣是集体性的，我们追寻艺术活动动态用的就不是两只眼睛，而是二十只、四十只眼睛。一个人错过的事情，另外一个人会帮他注意到；况且，因为我们带着差不多像运动场上的争强好胜之心，幼稚而调皮地想要在新知识以及对最新情况的了解上超越别人。
>



> 此前我曾经两次到过巴黎，对这座永不疲倦的城市只有粗浅的了解。不过，我知道，有谁年轻时在那里生活过一年的话，他终其一生都会带着无可比拟的幸福回忆想到它；我知道，没有任何地方能有和这座城市一样的氛围，能让人在充满活力的感觉中感受到它那青春气息：它会接受任何人，又不会对任何人追根究底。
>



> 最睿智的学说、最美好的榜样，同时又让人感到自由和创造活力；它对每个人都敞开心扉，在这毫不顾惜的慷慨付出中却变得越发富有。

> 一个人得先知道柏林是怎么回事，经历过德国那种心甘情愿的奴性——这里有着棱角分明、被痛苦地打磨出来的阶层意识——才会真心爱上巴黎。
